Summary

1.Title of the paper A Robust Quality Metric for Noisy Incomplete Networks
2. Noise Model - How did they perturb the network (did they provide an algorithm). Uniform noise model, Crawled noise model, Censored noise model
3. What data sets did they use (also the number of edges and vertices) Both synthetic and real world (LFR Benchmark – nodes=1000, Football, ca-CondMat and ca-HepPh, Email Enron)
4. What property/measures of network are they looking at Vertices, edges, permanence (ﬁrst unifying metric to handle different accuracy aspects of noisy, incomplete networks), closeness centrality and PageRank
5. How do they measure the difference in results -For each network, apply noise models at different noise levels (0% to 30%), and create ten networks for each case on which the results are averaged. --For the varying noise levels, compute the value of the network features for each network in the set as well as the values of permanence, modularity, closeness centrality and PageRank. --Then compute the Pearson correlation coefﬁcient of these values
6. Are there theoretical results--write the theorems Baseline sampling algorithms
7. What are the main conclusions--find it in the abstract and the conclusion Proposal:
-Permanence is an excellent indicator of how a network changes under noise
-Vertices with high permanence are very effective seeds from message broadcast
-Sampling based on high permanence vertices can retain the cluster structure of the network
Conclusion:
Permanence is a robust metric for evaluating and analyzing noisy incomplete networks
